    Field weakening nonlinear control of a separately excited DC motor

    Conference Paper · February 2004

    Abstract
    This paper presents three field weakening nonlinear control schemes for a separately excited DC motor. The first controller is designed using the feedback linearization technique. However, since this technique requires the exactly knowledge of the model and the parameters of the system, other robust controllers need to be designed. Hence, a Corless/Leitman type controller and a continuous nonlinear state feedback controller are proposed for this motor. Simulation…

    attached to the motor is tested. Experimental tests indicate that the feedback linearization
    controller is not robust to changes in the load. However, the Corless-Leitmann controller
    as well as the other two controllers are found to be robust to changes in the load attached
    to the motor.
